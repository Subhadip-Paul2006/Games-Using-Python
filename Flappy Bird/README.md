# 🐦 Flappy Bird Game (Python)

A simple Flappy Bird clone built using Python and Pygame.

---

## 🎮 Features

- Smooth bird animation
- Sound effects
- Collision detection
- Score tracking
- Classic Flappy Bird mechanics

---

## 📁 Files

- `Flappy_Bird.py` → Main game file
- `flappy_up.png`, `flappy_down.png` → Bird sprites
- `coin.png` → Coin image
- `*.wav` → Sound effects

---

## 🛠 Requirements

Python 3.x  
Pygame library

---

## ▶ How to Run

Install pygame:

```bash
pip install pygame
