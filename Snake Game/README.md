# 🐍 Snake Game (Python)

A classic **Snake Game** built using **Python**, created as part of my *Games Using Python* project series.

---

## 📌 Project Status
✅ **Complete**

---

## 🎮 Game Description
The player controls a snake that moves around the screen.  
The goal is to eat food, grow longer, and avoid colliding with the walls or itself.

---

## ✨ Features
- Smooth snake movement
- Food generation
- Score tracking
- Increasing difficulty (planned)
