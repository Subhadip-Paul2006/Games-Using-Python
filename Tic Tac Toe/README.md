# 🎮 Tic Tac Toe – Single Player (Python + PyGame)



This is a \*\*Single Player Tic Tac Toe game\*\* built using \*\*Python and PyGame\*\*, with \*\*database integration\*\* for storing game-related data.



---



## 📌 Basic Information



\- \*\*Game Name:\*\* Tic Tac Toe

\- \*\*Mode:\*\* Single Player

\- \*\*Language:\*\* Python

\- \*\*Framework:\*\* PyGame

\- \*\*Database:\*\* SQLite / PostgreSQL (configurable)



---



## 🎯 Game Objective



The objective of the game is to align \*\*three identical symbols\*\*:

\- Horizontally

\- Vertically

\- Diagonally



before the game ends.



---



## 🎮 Playing Rules



1\. The game is played on a \*\*3×3 grid\*\*.

2\. The player clicks on an empty cell to make a move.

3\. Each cell can be used only once.

4\. A win is declared if three symbols align.

5\. If all cells are filled and no win is formed, the game ends in a \*\*draw\*\*.

6\. The game can be restarted after completion.



---



## 🛠️ Requirements



\- Python 3.x

\- PyGame

\- Required libraries listed in `requirements.txt`



---



## ▶️ How to Run the Game



```bash
pip install -r requirements.txt
python main.py
```
